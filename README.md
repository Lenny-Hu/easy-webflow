# gulp-demo
使用gulp构建以vue为主，不使用前端路由的项目

## 目标

- sass => css
- es6|7 => js
- jade => html
- typescript => js
- css lint
- js lint
- html lint
- 开发服务器、自动重载、动态插入样式 Browsersync
- 根据配置文件启动开发服务或者构建生产环境资源

## 开发环境
---

`npm run dev`

> 将所有资源编译或复制到.tmp目录中，同时启动开发服

### css


## 生产环境
---

`npm run prod`

> 将所有资源打包压缩优化后到dist目录中，dist目录作为发布在生产环境的根目录
